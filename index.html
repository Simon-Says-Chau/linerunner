Line Runner

In this project I created a mini puzzle/rhythm game with embedded systems. The player needs to move vertically up with a reoccurring pattern. As I was creating the game, I had unintentionally added a different feature which in turn affected the final version of the game. I controlled the LEDs to have a set speed at different levels (3 levels in total). The rules of the game is to traverse the LED matrix from the bottom to the top while not touching the reoccurring pattern. I programmed with ATmega 1284 8-bit AVR Microcontrollers. After completion, the LED matrix would light up in green.
